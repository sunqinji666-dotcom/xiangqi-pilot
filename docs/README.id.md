# XiangqiPilot

> Kokpit permainan papan untuk macOS: mengamati dulu, memverifikasi, lalu bertindak hanya saat aman.

[简体中文](../README.md) · [English](README.en.md) · [日本語](README.ja.md) · **Bahasa Indonesia**

Dibuat oleh **Jacksun (孙秦吉)** · [qinji@jack-sun.com](mailto:qinji@jack-sun.com)

XiangqiPilot membantu catur Tiongkok dan Gomoku: mengenali jendela target, mengalibrasi papan, memvalidasi posisi, dan memberi saran mesin. Klik bersifat opsional dan hanya dilakukan setelah pemeriksaan jendela, geometri papan, posisi, langkah, serta kesegaran bingkai sebelum dan sesudah tindakan. Ini bukan skrip otomatisasi buta.

Memerlukan macOS 14+ dan Xcode 26+; Apple Silicon disarankan. Jalankan `scripts/test.sh`, lalu bangun secara lokal dengan `scripts/setup-local-signing.sh` dan `scripts/build-app.sh`. Berikan izin perekaman layar dan aksesibilitas hanya saat diperlukan.
